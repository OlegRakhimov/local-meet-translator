# Landing Page Package (Local Meet Translator)

This folder contains a minimal, fast landing page suitable for Product Hunt.

## What you need to edit (required)
Open `index.html` and replace these placeholders:

- `__CHROME_WEB_STORE_URL__`
- `__EDGE_ADDONS_URL__`
- `__DOWNLOAD_MAC_LINUX_URL__`
- `__GITHUB_REPO_URL__`
- `__SUPPORT_EMAIL__`

Also replace `downloads/bridge-windows.zip` with your actual Windows bundle.

## Deploy options

### Option A — GitHub Pages (fastest)
1. Create a repo (or use your existing repo).
2. Put the landing files into `/docs`.
3. In GitHub: Settings → Pages → Deploy from branch → `/docs`.

### Option B — Vercel / Netlify
- Drag-and-drop the folder or connect the repo.

## Local preview
Open `index.html` in your browser.

Tip: keep the first screen (hero) simple: 2 buttons + 1 video.
